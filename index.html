Présentation

Bienvenue sur ScoreCenter - Ours Toulouse. Ce site présente les résultats de l'équipe de football américain des Ours de Toulouse, évoluant en première division nationale avec comme objectif cette saison : le casque de diamant !

Les données affichées proviennent d'une base MySQL en ligne, accessible via une API Node.js déployée sur Render.
